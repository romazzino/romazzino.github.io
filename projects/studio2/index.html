mineral structure gridshell

07/2019

a shell structure that derives its geometry from the mineral herbertsmithite, which has a hexagonaly ordered arrangement of atoms. This geometry, transferred to a supporting structure, offers advantages for the load-bearing capacity and thus the lightness of the structure.

This system consists of two repetitive standardized parts, a triangle and an octahedron, which are connected with stiff nodes that are unique for each position in the shell.

the parametric model for this project covers topics such as vector optimization, stress testing, sun analysis.

the system is capable of creating gridshells based on only a perimeter curve as input as well as desired variables for the gridshell like panel size, resolution, and height of the structure i.e. .

plugins: weaverbird, kangaroo, karamba3d, ladybug, metahopper.
